# HeartMatch 心动匹配

遇见心动，开启你的浪漫之旅。

## 产品介绍

HeartMatch 是一款专为寻找真诚关系的单身人士设计的社交应用。通过智能匹配算法，让你更高效地发现志趣相投的那个人。

## 主要功能

### 发现与匹配
- **滑动匹配** - 左滑跳过，右滑喜欢，上滑超级喜欢
- **智能推荐** - 根据你的喜好和互动，算法每日推荐最合适的对象
- **双向喜欢即配对** - 两个人互相喜欢即可开始聊天

### 动态分享
- **图文动态** - 发布生活点滴，展示真实的自己
- **话题标签** - 参与热门话题，发现同好
- **互动功能** - 点赞、评论、收藏、分享

### 即时通讯
- **匹配聊天** - 和喜欢的人实时对话
- **消息历史** - 保留每一份珍贵回忆

### 个人主页
- **资料完善** - 展示照片、个人信息、兴趣爱好
- **认证体系** - 实名认证、学历认证、职业认证，让你更安心

## 如何开始

### 注册账号
1. 下载并打开 HeartMatch
2. 使用手机号注册
3. 填写验证码完成验证
4. 设置登录密码或使用验证码登录

### 完善资料
1. 上传清晰的头像照片（至少1张）
2. 填写基本信息：昵称、性别、生日、城市
3. 可选完善：职业、学历、身高、体重、兴趣爱好
4. 进行认证，提升可信度

### 开始匹配
1. 进入"发现"页面浏览推荐用户
2. 右滑喜欢，左滑跳过
3. 收到互相喜欢通知，开始聊天
4. 通过动态了解对方更多

- 你的位置信息仅用于推荐匹配，不会公开
- 敏感操作需要验证身份
- 发现虚假账号可一键举报
- 黑名单功能，屏蔽不想联系的人
